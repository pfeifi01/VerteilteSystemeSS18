[1] put your ".pem" file here
[2] execute shell from this folder (i used git-bash)
[3] 'chmod 400 XY.pem' in shell
XY is the name of your file
[4] 'chmod +x createNewInstance.sh'
'chmod +x stopAll.sh'
'chmod +x terminateAll.sh'
[5]'./createNewInstance.sh' for new instance
[6]'./stopAll.sh' to stop all created instances
[7]'./terminateAll.sh' to terminate all created instances
